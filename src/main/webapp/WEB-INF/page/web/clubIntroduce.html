酷撩学院平台介绍

  • 酷撩学院成立于2012年，前身隶属于香港顶峰集团有限公司，2017年由猴赛雷（广州）文化传媒有限公司正式引入中国内地。

品牌历程

  • 2011年 随着经济的发展，香港社会普遍存在的婚恋情感问题也日益突出，许多年轻人因感情问题备受困扰。当时香港首批情感专家、香港顶峰集团有限公司总经理陈鸿锦先生和香港中文大学心理学系的几个高校同仁在一次心理论坛交流会上谈及到年轻人的情感问题时，一致认为心理学家有责任和义务帮助广大青年男女解决问题。

    2012年3月陈鸿锦先生联合香港中文大学心理学系、香港科技大学人文社会科学学院等几个高校内的好友组建了一支具有理论、实战经验丰富的心理专家团队，在香港正式成立酷撩学院，专注于为用户提供情感咨询、解决情感困惑。

    2013年酷撩学院战略聚焦，专注于为男性用户提供追女生技巧、情感教育咨询和自我魅力提升服务。不仅提供了线上课程，也开设了线下核心课程，同时还在内地北上广举办高峰论坛，受到行业的广泛关注。

    2015年酷撩学院荣获多项大奖，包括中国情感导师奖，香港最佳情感成长奖、香港时尚魅情奖等大奖，被国内外各大媒体争相报道。

    截止到2016年，酷撩学院已成功帮助上万逾名学员追到自己的心仪的女生。

    2017年，为进一步拓展内地市场，在广州成立了猴赛雷文化传媒有限公司，正式进军内地，继续专注于为男同胞提供追求女生技巧知识服务，情感问题解决方案以及提升个人魅力。
